# MCP Stdio Ruby Server

A simple Model Context Protocol (MCP) server implemented in Ruby using the `mcp` SDK.

## Features

- **greet**: A simple tool that echoes back a message.

## Setup

1.  **Install Ruby**: Ensure you have Ruby 3.0+ installed.
2.  **Install Dependencies**:
    ```bash
    bundle install
    ```

## Running the Server

To run the server locally via stdio:

```bash
bundle exec ruby main.rb
```

## Testing

Run tests with RSpec:

```bash
bundle exec rspec
```

## Linting

Run RuboCop:

```bash
bundle exec rubocop
```
