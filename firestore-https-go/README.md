# benchmark-go

This is a Go project that serves as a backend service for a distributed system, specifically a benchmark for the A2A protocol using ADK (Agent Development Kit).

## Project Structure

- `main.go`: The main application file.
- `main_test.go`: Unit tests for the main application.
- `go.mod`: Go module definition file.
- `go.sum`: Go module checksum file.
- `Makefile`: Contains commands for building and running the project.
- `README.md`: This file.
- `GEMINI.md`: Additional project context and guidelines for the Gemini CLI.

## Getting Started

To build and run this project, use the following commands:

```bash
make build
make run
```
